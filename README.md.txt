# Agricultural Pest Detection using YOLOv8

This project detects agricultural pests using a YOLOv8 deep learning model.

## Dataset
- 5 pest classes
- 400 images

## Model
YOLOv8 trained for 50 epochs using Google Colab.

## Results
mAP@0.5 = 95%

## Technologies Used
- Python
- YOLOv8
- Roboflow
- Google Colab
